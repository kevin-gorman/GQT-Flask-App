Welcome to GQT

Genotype Query Tools (GQT) is software for indexing and querying large-scale genotype data sets. This website is a web interface for the software specifically designed for querying one of the most commonly used data sets, '1000 Genomes Project' Phase 3 autosomal data sets

This data set is comprised of genotypes for 2504 individuals across 26 populations. The populations and their encodings are the following:
